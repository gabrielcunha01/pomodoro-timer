# ⏱️ Pomodoro Timer Pro

Aplicação de gerenciamento de tempo focada em produtividade e experiência do usuário.

### 🛠 Tecnologias e Conceitos:
- **Timing Functions:** Lógica de contagem regressiva com `setInterval`.
- **Dynamic UI:** Atualização do título da aba do navegador para monitoramento de tempo.
- **Clean Design:** Interface minimalista e intuitiva.
